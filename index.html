The five tabs
Dashboard
Overall progress, firm milestone countdowns, and tasks due in the next 30 days. Reflects any date changes immediately — preview dates show an amber badge.
Schedule
Full list of all 46 tasks. Search, filter by phase or status, update task status inline with the dropdown, and export to CSV.
Calendar
Monthly view from March–November 2026. Navigate with Prev/Next. Task chips appear on their start dates, color-coded by phase. ★ marks firm dates; ■ marks holidays.
🔔 Reminders
Configure email reminder timing and preferences, then generate a pre-filled email via your local email client (Outlook, Gmail, etc.). Nothing is sent automatically.
Dates
Adjust the two anchor dates — Live Event and Webinar. Anyone can explore changes in preview mode. Saving requires the admin PIN. The Live Event is a 3-day event and must start on a Monday, Tuesday, or Wednesday so all days fall on workdays — the app will block any other choice.
How date cascading works
↳ event
This task's dates shift automatically when the Live Event anchor date changes. 20 tasks cascade this way.
↳ webinar
This task shifts with the Webinar anchor date. 4 tasks cascade this way.
fixed
This task has a fixed date that does not change when anchors are adjusted. 22 tasks are fixed.
★ Firm dates
Four confirmed milestones that should not move without a major decision: Survey soft launch (Apr 1), Survey close (Jul 31), Webinar (Oct 6), Live Event (Oct 27).
Preview vs. official dates
Preview mode
Anyone can move the anchor dates in the Dates tab to explore how the schedule would shift. An amber preview tag appears on affected dates across all tabs. Nothing is committed.
Official dates
The saved dates the whole team sees. Shown with a green ✓ Official tag. Only updated when an admin enters the PIN and clicks Save.
Discard preview
Click "Discard all" in the Dates tab, or "Discard preview" in the yellow banner, to revert to official dates without saving.
Admin PIN
Who needs it
Only the 1–2 people authorized to commit official date changes. Everyone else can explore freely without a PIN.
How it works
Click 🔒 Save as official dates in the Dates tab. A PIN prompt appears. Once entered correctly, admin mode stays active for the rest of your browser session — you won't be asked again until you lock or close the file.
Task statuses
● Completed
Task is done. No longer counted in upcoming reminders.
● In Progress
Currently active. Shows in Dashboard progress bars as a faded bar segment.
● Not Started
Default state. No action yet.
● Blocked
Task is stuck. Triggers a warning in the Reminders tab digest if that alert is enabled.
Version stamp
What it shows
The small gray date below the title bar shows when the file was last updated. If your version doesn't match what the team was told, close and reopen the file from the shared drive folder.
Update banner
When you open a version you haven't seen before, a dark blue banner appears at the top of the page. Click Got it to dismiss it — your browser will remember and won't show it again for this version.
How it's updated
Whoever manages the file updates the FILE_VERSION line near the top of the script each time a new version is distributed.
